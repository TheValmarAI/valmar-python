# Security Policy

## Reporting a Vulnerability

Do not open public issues for security vulnerabilities.

Report suspected vulnerabilities to the Valmar maintainers at security@getvalmar.com with:

- affected package and version
- deployment context
- reproduction steps
- observed impact

We will acknowledge reports and coordinate fixes privately before public disclosure when appropriate.

## Secrets

Never include Valmar API keys, deployment credentials, `.env` files, logs with bearer tokens, or customer data in issues, pull requests, examples, or tests.
